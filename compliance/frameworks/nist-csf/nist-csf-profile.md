## Functions

NIST CSF 2.0 organizes outcomes into six functions:

| Function | Purpose |
| --- | --- |
| Govern (GV) | Establish and monitor the cybersecurity risk strategy and governance |
| Identify (ID) | Understand assets, risks, and the business context |
| Protect (PR) | Implement safeguards to limit or contain impact |
| Detect (DE) | Find and analyze possible cybersecurity events |
| Respond (RS) | Take action on detected incidents |
| Recover (RC) | Restore capabilities affected by incidents |

## Implementation Tiers

| Tier | Name | Description |
| --- | --- | --- |
| 1 | Partial | Ad hoc, reactive risk management |
| 2 | Risk Informed | Risk practices approved but not org-wide |
| 3 | Repeatable | Formal, consistent organization-wide practices |
| 4 | Adaptive | Continuous improvement and adaptation |
