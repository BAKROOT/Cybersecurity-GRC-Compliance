# Cybersecurity GRC Compliance

A complete, ready-to-use **Governance, Risk, and Compliance (GRC)** program repository. It provides the policies, charters, registers, control matrices, framework mappings, audit evidence trackers, and templates needed to stand up or mature an information-security GRC function.

## What is GRC?

- **Governance** - the structures, roles, and policies that direct and control the security program.
- **Risk** - identifying, assessing, treating, and monitoring information-security risk.
- **Compliance** - demonstrating adherence to regulatory, contractual, and framework obligations (ISO 27001, SOC 2, NIST CSF, etc.).

## Repository Structure

| Path | Purpose |
| --- | --- |
| governance/ | Program direction and authority: charters, policies, standards, procedures |
| risk/ | Risk management framework, methodology, templates, and the living risk register |
| compliance/ | Framework mappings (ISO 27001, SOC 2, NIST CSF) and a cross-framework control crosswalk |
| audit/ | Internal and external audit, evidence tracker, and audit report template |
| templates/ | Reusable blank document templates |

Each folder contains its own README.md with a full, path-by-path map of its files.

## How to Use This Repository

1. Adopt the governance docs - tailor the policies and charter to your organization, then route them for approval.
2. Stand up risk management - use the methodology and template to populate the risk register.
3. Map your obligations - pick your framework(s) under compliance/frameworks/ and complete the applicability/profile docs.
4. Track evidence - record control evidence in audit/evidence/evidence-tracker.csv ahead of audits.
5. Iterate - review policies at least annually; update the risk register continuously.

## Document Lifecycle

| Stage | Description |
| --- | --- |
| Draft | Authored and under internal review |
| Review | Reviewed by control/process owners |
| Approved | Signed off by accountable owner |
| Published | In force and communicated |
| Retired | Superseded or no longer applicable |
