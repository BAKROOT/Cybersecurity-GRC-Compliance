# Templates

Reusable, blank templates for creating new GRC documents in a consistent format.

## Files

| Path | Purpose |
| --- | --- |
| policy-template.md | Standard structure for a new policy |

## Related Templates Elsewhere

| Path | Purpose |
| --- | --- |
| risk/templates/risk-assessment-template.md | Assess a new risk or initiative |
| audit/templates/audit-report-template.md | Document an audit engagement |

## How to Use

1. Copy the template into the correct domain folder.
2. Replace the placeholders (document ID, owner, dates).
3. Set the status to Draft and route for review and approval.
4. Update the relevant folder README and the control crosswalk if the document maps to a control.
